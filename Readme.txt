Chicago Social Hub

Setting up the environment:
	1. Run pgAdmin server
	2. Run elasticserver by running elasticserver.bat file from cmd
	3. Run ChicagoSocialHub-Yelp.ipynb file
	4. Run divvy_real-time_status.pynb file
	5. Run divvy_trips.ipynb file

Running Backend:
 Open the backend folder from cmd and execute the following commands
	1. 'npm install'
	2. 'node server.js'	 

Running Frontend:
 Open the frontend folder from cmd and execute the following commands
	1. 'npm install'
	2. 'ng serve'
